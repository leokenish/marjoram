텍스트 크기 조절
루트 엘리먼트(html)의 폰트 사이즈가 기준(rem)이 되며 원하는 텍스트 크기에 따라 small, medium, large, xlarge 중 하나의 클래스를 적용 하시면 됩니다.
<div class="xlarge">1.5 rem이 적용된 매우 큰 글씨체 입니다.</div>
<div class="large">1.2 rem이 적용된 큰 글씨체 입니다.</div>
<div class="medium">1 rem이 적용된 중간 글씨체 입니다.</div>
<div class="small">0.8rem이 적용된 작은 글씨체 입니다.</div>

팁 보여주기
텍스트에 마우스 오버 시 팁을 보여주려면 자식 태그에는 tooltiptext 클래스를 사용 하고 부모 태그에는 tooltip 클래스를 적용 하면 됩니다.
<p class="tooltip">팁을 보시려면 마우스를 텍스트위에 올려 주세요
<span class="tooltiptext">이곳에 팁 내용이 들어갑니다.</span>
</p>

팁을 보시려면 마우스를 텍스트위에 올려 주세요. 이곳에 팁 내용이 들어갑니다.

텍스트 입력창 만들기
prepend, append, middle 태그를 input 태그와 input_group클래스로 그룹지어 사용합니다.
각 태그별로 prepend는 input 태그 앞쪽, append는 input 태그 뒤쪽, middle 태그는 input 태그 가운데 위치하며
각 태그를 혼용하여 순차적으로 사용도 가능합니다. 또한 태그에 클래스를 입력하여 색상을 변경 할 수 있습니다.
<div class="input_group">
 <prepend>전화번호</prepend>
 <input type="text" name="phone" size="30" maxlength="20" placeholder="'-'를 제외하고 입력" />
</div>
<div class="input_group">
 <input type="text" name="dollor" size="20" maxlength="10" placeholder="숫자만 입력" />
 <append>$</append>
</div>
<div class="input_group">
 <input type="text" name="mail" size="20" maxlength="10" value="maple" />
 <middle>@</middle>
 <input type="text" name="domain" size="10" maxlength="20" value="kbs.co.kr" />
</div>
<div class="input_group">
 <prepend class="red">red</prepend>
 <input type="text" name="phone" size="30" maxlength="20" placeholder="클래스 속성으로 색깔 변경 가능" />
 <middle class="green">green</middle>
 <append class="blue">blue </append>
</div>

선택 창 만들기
prepend 태그를 라벨로 하여 기본적인 select, option 태그를 사용하면 됩니다. option 태그에 클래스를 두어 색상 변경도 가능합니다.
<div class="input_group">
 <prepend>color</prepend>
 <select name="color">
  <option value="0">선택</option>
  <option value="red">RED</option>
  <option value="blue">BLUE</option>
  <option value="yellow">YELLOW</option>
  <option value="green">GREEN</option>
 </select>
</div>
<div class="input_group">
 <prepend>color</prepend>
 <select name="color">
  <option value="0">선택</option>
  <option class="red" value="red">RED</option>
  <option class="blue" value="blue">BLUE</option>
  <option class="yellow" value="yellow">YELLOW</option>
  <option class="green" value="green">GREEN</option>
 </select>
</div>
